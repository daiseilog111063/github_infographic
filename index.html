🎓Git & GitHub の成長ステップ

初級: 基本操作

  • push/pull の基本
  • GitHub アカウント作成
  • Issue 作成・コメント
中級: チーム作業
  • ブランチ戦略の理解
  • Pull Request の活用
  • コンフリクト解決
  • コードレビュー
上級: 高度な操作
  • rebase/cherry-pick
  • bisect/blame
  • GitHub Actions
  • GitHub API 活用
エキスパート: プロセス設計
  • 組織に最適な Git ワークフロー構築
  • 自動化パイプライン設計
  • ガイドライン・研修制度整備
  • 高度なマージ戦略

このインフォグラフィックについて

このコンテンツはリバースメンター制度 Git/GitHub セッション教材をベースに、視覚的な学習体験として再構成したものです。

各ページは印刷して職場に掲示したり、オンラインでチームメンバーと共有したりすることができます。

  • 基本から順に学んでいくことをおすすめします
  • 実際にコマンドを試しながら読むとより理解が深まります
